# Monitoring LLM and Agentic AI Systems in Banking

Large language models are increasingly being deployed in banking workflows,
from compliance assistants to knowledge retrieval systems.

Traditional model risk frameworks were designed for deterministic statistical models.
Generative AI systems behave differently and require continuous operational monitoring.

## Key Challenges

• Silent performance degradation  
• Prompt changes altering model behavior  
• Vendor model updates  
• Retrieval pipeline misconfiguration  

## Governance Implications

Monitoring LLM systems requires extending model risk governance across the three lines of defense.
